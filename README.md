# Blog Platform Frontend

A modern and responsive frontend for the Blog Platform application built using HTML, CSS, and JavaScript. It allows users to register, log in, create blogs, view blogs, and comment on blog posts by communicating with the backend REST API.

## 🚀 Features

- User Registration
- User Login
- View All Blogs
- Create Blog
- View Blog Details
- Add Comments
- Responsive Design
- REST API Integration

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

## ▶️ Run Locally

Clone the repository:

```bash
git clone https://github.com/Stormboomop/blog-platform-frontend.git
```

Open the project folder:

```bash
cd blog-platform-frontend
```

Start a local server:

```bash
python -m http.server 5500
```

Open:

```
http://localhost:5500
```
